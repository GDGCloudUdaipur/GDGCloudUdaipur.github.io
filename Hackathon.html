🧩 Tracks & Problem Statements

1. Clone & Innovate

Reinvent a Business Model

Objective: Choose any existing digital business (e.g., Zomato, Paytm, Airbnb, Urban Company) and clone its core functionality using Google Cloud — but with a unique, innovative twist.

Examples of Innovation: Voice-first interface, Smart pricing by AI, Integration with local government schemes, Personalization via user behavior analysis.

GCP Tech Ideas: Firebase, Cloud Run, Vertex AI, BigQuery

2. Build One-of-a-Kind AI Tools

Unleash AI Creativity

Objective: Create an AI-powered application that tackles a unique or overlooked problem. The more creative, the better.

Examples: AI for farmer crop health advice, AI for journaling mental health insights, A “reverse search engine” – input an outcome, get pathways, Resume parser & job-fit predictor.

Use Cases: Vertex AI, PaLM APIs, Natural Language, Cloud Vision

3. Civic Tech: Build for Better Governance

Tech for Public Good

Objective: Develop tools that improve government transparency, public access, or citizen participation.

Problem Angles: Local budget visualization tools, Civic issue reporting via mobile + geotagging, Chatbot for government services using Dialogflow, Community voting on local projects.

Tools to Explore: Google Maps API, Firebase, Dialogflow, Cloud Functions

4. Smart Business Data Analyzer

Now with Direct Email Data Fetching!

Objective: Design a secure web portal that processes multi-format business data fetched directly from emails and converts it into editable, structured tables, with final output submitted to an API.

Data to be fetched from Email: PDF, Excel, CSV, Text, Images (receipts/invoices) from email attachments or body using the Gmail API.

Workflow: Use OCR + NLP to extract info from fetched data, Display it in editable table format, Secure login portal, On submit, send data to external API.

Google Tools to Consider: Document AI, Vision API, Gmail API, Firebase Auth, Firestore

📦 Submission Guidelines

You will need to submit all of the following via our official submission Google form.

  • Demo Video: A 2-3 minute video uploaded to YouTube (set to public or unlisted). It should explain the problem, your solution, and walk through the demo.
  • Code Repository: A Google Drive folder containing all your source code. The sharing setting must be "Anyone with the link can edit."
  • Pitch Deck: A 4-5 slide presentation on Google Slides (with view access). It must cover the problem, solution, innovation, architecture, and the Google tech you used.

🏅 How to Get Bonus Points

  • Deep Google Tech Integration: Creative and extensive use of GCP services or Google APIs.
  • Live Deployment: A working live demo hosted on a GCP service like Cloud Run or Firebase Hosting.
  • Polished UI/UX: A project with a clean, intuitive, and smooth user experience will stand out.

🤔 Frequently Asked Questions

🙋‍♀️ General

Who is eligible to participate?

The hackathon is open to students, professionals, entrepreneurs, and hobbyists from across India. All skill levels are welcome!

Is there a team size limit?

Yes! You can participate solo or form a team of up to 3 members.

Is there a registration fee?

No, CloudCraft 2025 is a completely free event, powered by Google Developer Groups Cloud Udaipur.

Do I need to attend in person?

Yes, if you are among the finalists you might be asked to attend the event in person for the final presentation and to collect your prize if you win.

What is the Code of Conduct?

All participants must abide by the GCC Code of Conduct. We are committed to a respectful, inclusive, and collaborative environment.

🛠️ Technical & Support

Do I need prior experience with Google Cloud?

Not at all! We encourage learning with Gemini and the use of AI to build projects.

Can we start coding before the official start date?

Yes, the Hackathon has officially started and you can start coding and submit it before the submission deadline.
